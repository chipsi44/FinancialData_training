
create a docker network
docker network create my-network

get firefox selenium grid 
docker pull selenium/standalone-firefox 

run the selenium grid in my network
docker run -d --network my-network --name selenium-container -p 4444:4444 selenium/standalone-firefox:latest               

build docker image of my app for airflow
docker build -t monapp:1.0 . 

run the app for
docker run -d --network my-network --name monapp-container -p 8080:8080 monapp:1.0                                         

Copy the file from the docker to the local users ! 
docker cp monapp-container:/app/dags/data/financeYahoo_dataframe.csv ./data
